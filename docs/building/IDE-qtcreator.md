# Qt Creator 开发
## 前提
- 请参考 [**VS Studio**](./docs/IDE-vsstudio.md) 安装VS studio环境

## 打开项目
- 文件-》打开文件或项目
- 选择项目根目录[CMakeLists.txt]即可

## 构建项目
- 打开项目之后，应该会生成几套编译配置
- 选择Release构建模式构建即可

## 运行项目
- 输出目录位于：`out\build\x64-Debug`，或者 点击配置管理器=》查询CMake变量=》找到CMAKE_INSTALL_PREFIX
- 点击运行 `OkEDU-Classroom-Desktop.exe` 即可。
